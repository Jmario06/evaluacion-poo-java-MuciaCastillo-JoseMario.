# Sistema de gestión de tareas escolares
## El proyecto es de la creación de una aplicación sobre gestión de tarea, donde el docente puede asignar tareas y marcarlas como calificada.


### Características
- Creación de tareas con ID.
- Marca tareas pendiebtes  y guarda fecha de entrega.
- Valida datos vacíos, IDS repetidos y tareas inexistentes.


### REQUISITOS
##### 
Tener unicamente Java JDK 21
